# Phoenix Panel Roadmap

Phoenix Panel is built one step at a time.

Every version has a purpose.

We don't release software because it is finished.

We release software because it is ready.

---

- Basic dashboard
- Authentication
- Cloudflare Workers integration
- Initial settings
- Responsive layout
- Dark mode

Status:

🟡 In Progress

Future ideas.

---

# Development Principles

Every feature must:

- Solve a real problem.
- Keep the project simple.
- Respect user privacy.
- Improve the user experience.

If a feature does not follow these principles,

it does not belong in Phoenix Panel.
